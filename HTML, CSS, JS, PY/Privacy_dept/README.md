# Privacy Dept Visualizer Setup

## requirement software
<ul>
<li>Python 3.9.18 or Anaconda3-2023.09-0-Windows-x86_64</li>
<li>MySQL server 5.0</li>
<li>MySQL Tool for 5.0</li>
</ul>

<ol>

</ol>
